2. Template Tags

Tag Description Example

3. Template Filters

String Filters

Filter Description Example
lower Lowercase {{ name|lower }}
upper Uppercase {{ name|upper }}
title Title Case {{ name|title }}
capfirst Capitalize first {{ name|capfirst }}
truncatechars Truncate characters {{ text|truncatechars:50 }}
truncatewords Truncate words {{ text|truncatewords:10 }}
striptags Remove HTML {{ html|striptags }}
slugify Slug format {{ title|slugify }}
linebreaks Newlines to <p> {{ text|linebreaks }}
linebreaksbr Newlines to <br> {{ text|linebreaksbr }}
cut Remove characters {{ text|cut:" " }}

Number Filters

Filter Description Example
add Add value {{ num|add:5 }}
divisibleby Check divisibility {{ num|divisibleby:2 }}
floatformat Format float {{ price|floatformat:2 }}
filesizeformat Human readable size {{ size|filesizeformat }}

Date Filters

Filter Description Example
date Format date {{ date|date:"Y-m-d" }}
time Format time {{ time|time:"H:i" }}
timesince Time since {{ date|timesince }}
timeuntil Time until {{ date|timeuntil }}

List Filters

Filter Description Example
length Get length {{ list|length }}
first First item {{ list|first }}
last Last item {{ list|last }}
join Join items {{ list|join:", " }}
slice Slice list {{ list|slice:":3" }}
random Random item {{ list|random }}
dictsort Sort by key {{ list|dictsort:"name" }}

Default/Boolean Filters

Filter Description Example
default Default value {{ var|default:"N/A" }}
default_if_none Default if None {{ var|default_if_none:"N/A" }}
yesno Boolean mapping {{ flag|yesno:"Yes,No" }}

Safe/Escape Filters

Filter Description Example
safe Mark safe HTML {{ html|safe }}
escape Escape HTML {{ text|escape }}
escapejs Escape for JS {{ text|escapejs }}
urlencode URL encode {{ text|urlencode }}
urlize URLs to links {{ text|urlize }}

Quick Reference

SYNTAX:
  Tags:       {% tag %}...{% endtag %}
  Variables:  {{ variable }}
  Filters:    {{ variable|filter:arg }}
  Comments:   {# single line comment #}
              {% comment %}multi line{% endcomment %}

COMMON PATTERNS:
  {% if user.is_authenticated %}Welcome!{% endif %}
  {% for item in items %}{{ item }}{% empty %}None{% endfor %}
  {% url 'view-name' object.id %}
  {% static 'css/style.css' %}
  {{ name|title|truncatewords:5 }}
  {{ date|date:"Y-m-d H:i" }}
  {{ price|floatformat:2 }}
  {{ items|length }}
